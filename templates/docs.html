SteamRate API dokumentation v1

Endpoint

make GET requests to:

api/search?q=[steamid]

Request parameters:

q: search with your steamid


Response

{
    "average": 
    "result":  
}

Hitta ditt Steamid

Steamid hittar du på steam där du loggar in och sen klickar på "visa profil"

Sen kan du se ditt steamid i URL

# These code snippets use an open-source library. http://unirest.io/python

response = unirest.get("http://currenturl/api/search?q=STEAMID"
                           headers={ "Accept": "application/json" })

respone.body # to extract the json data response

Javascript code example:

# These code snippets use jquery library. http://api.jquery.com/

var ENDPOINT = "http://currenturl/api/search";
var q = $("input[name=q]").val();

$.ajax({
		url: ENDPOINT,
		data: {q: q},
		success: showResult,
		dataType: "json"
	});

function showResult(response){
		console.log(response);
}
